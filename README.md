## Running the project

First install Nodejs:

```shell
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

Then clone the project:

```shell
git clone https://github.com/morphisjustfun/SSTree
```

Install dependencies:

```shell
npm install
```

Run the project:

```shell
npm run start
```
